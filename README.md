# Go/Fiber Auth Server
Authentication server using Go/Fiber V3 for my study.
 
## Endpoints

| Method | PATH  |
| :--- | :--- |
| GET | /ping |
| POST | /register |
| POST | /login |

## Dotenv Configuration
```
DB_USER= [DATABASE USERNAME]
DB_PASS= [DATABASE PASSWORD]
DB_NAME= [DATABASE NAME]
```
